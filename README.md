# Python Practice Programs

This repository contains beginner-level Python programs created while practicing programming concepts.

## Programs Included

1. Guess Number Game
2. List Manager
3. Password Generator
4. Simple Calculator

## Concepts Practiced

- Loops
- Conditional Statements
- Functions
- Lists
- User Input Handling
- Basic Logic Building

## Technologies Used

- Python 3

## How to Run

1. Install Python
2. Download the repository
3. Run any program using:

python filename.py

Example:

python calculator.py

## Purpose

This repository is created to practice Python fundamentals and improve problem-solving skills.
